Proyectos realizados en el Bootcamp:

Primer ejercicio (Formulario de banco):

Realizamos entre tres personas, al poco de comenzar el Bootcamp, un formulario para web de banco con HTML5, CSS3 y Javascript.
Se le intentó dar una estética realista, fijándonos en la app del BBVA. Se trabajó sin métodos, haciendo todas las validaciones con algoritmia de Javascript puro.

Cines Paradiso:

Presentación sobre una web completa realizada únicamente con Frontend. El reto estuvo, precisamente, en lograr la persistencia de los datos de compra y usuario sin nada de Backend, utilizando Storage para mantener los datos en el navegador.
En esta presentación se alterna la presentación de la app Web propiamente, con la explicación del código en algunos puntos importantes.

Asistencia en carretera:

En este caso, se nos solicitó una aplicación web hecha únicamente con Backend, realizando las vistas con Pug y EJS.
Se trata de una página hecha para una empresa ficticia de asistencia en carretera, con la posibilidad de registrarse para tener un seguimiento de las solicitudes, o poder pedir asistencia sin registro. Tiene geolocalización en tiempo real reflejada en un mapa para saber por dónde va la grúa.
Se presentan tambíen funcionalidades sólo accesibles a la cuenta del administrador de cara a gestionar las solicitudes (autorizarlas o rechazarlas), enviándole al cliente el número de seguimiento en cuanto se autoriza el servicio.

Proyecto final (Red Emancipatic):

Presentación del proyecto final MERN, realizado entre las ramas de Fullstack, UIX, Marketing Digital, Ciberseguridad y Data Science. En este caso, las compañeras de UIX y Marketing realizan la presentación del proyecto, enfocándose en llevar al cliente (real) las soluciones propuestas en un lenguaje accesible para este último.
El cliente solicitó una aplicación web que cubriese la necesidad de la Asociación Emancipatic para captar voluntarios y donantes. De esta manera, los voluntarios podían darse de alta, a la vez que los asociados podían solicitar ayuda de voluntarios para resolver los diversos problemas que las personas mayores pudieran tener con las nuevas tecnologías.
